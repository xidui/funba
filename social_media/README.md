# Social Media

Platform-specific content and publishing code lives here.

Current structure:

- `hupu/` — Hupu publishing implementation, forum normalization, and local runtime files
- `reddit/` — Reddit cookie import and login-state checks
- `twitter/` — X/Twitter cookie import and login-state checks
- `xiaohongshu/` — Xiaohongshu cookie import and creator login-state checks

Future platforms can live alongside it, for example:

- `discord/`
