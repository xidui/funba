# Game Analysis Review Profile

Use this profile for game-analysis SocialPosts created from a game-analysis Paperclip issue.

Apply the full generic checklist in `agents/content-reviewer/AGENTS.md`.

Extra emphasis:

- Read the analyst's `Story Signals` note in the Paperclip issue.
- Verify the chosen angle is defensible against the game's strongest P1 signals.
- Check source game IDs, metric links, ending related-link list, and image placeholders.
- Review the enabled image pool in article context. Normal game-analysis posts should generally have a strong multi-image pool and valid `[[IMAGE:slot=...]]` placements.
